# PySideChessBoard
A chess board widget for PySide6, similar to chessboard.js but for PySide.

## Features

Set board FEN\
Moves history (navigate with arrows)\
Legal moves highlighting\
Square highlighting (red, green, yellow, blue)\
Draw arrows\
